# 🎲 Dice Roller Simulator (C)

A simple Dice Roller Simulator written in C that generates a random number between 1 and 6, simulating a real dice roll.

---

## 🚀 Features
- Generates random dice values (1–6)
- User can roll multiple times
- Simple and beginner-friendly

---

## 🛠️ Technologies Used
- C Programming Language
- Standard Libraries: stdio.h, stdlib.h, time.h

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/dice-roller-c.git

2. Navigate to the folder:
   cd dice-roller-c

3. Compile the program:
   gcc main.c -o dice

4. Run the program:
   ./dice
